# Mi pimer modelo

Se requiere que modele el concepto de una persona.

Una persona posee:  

* nombre
* apellido
* número de documento de identidad
* año de nacimiento.
* edad (en función de su año de nacimiento)
La clase debe tener un constructor que inicialice los valores de sus respectivos atributos. Ojo, el atributo edad se calculará mediante el uso de un método.



#### PD:
No es necesaria ninguna salida por terminal. No es una aplicación de consola.

#### Requisitos:

Modelar la entidad con sus atributos
Se debe realizar los tests pertinentes para alcanzar un coverage mínimo del 70%
Entregable:
Enlace del repositorio de GitHub (enlace)
Diagrama de clase (enlace o captura de pantalla)
Captura de pantalla de la sección "Testing" de VSC donde se aprecia la cobertura de los tests.
